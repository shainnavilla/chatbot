# Tech Stack

This project uses the following technologies:

## Frontend
- **HTML5** - Single-page application structure 
- **Vanilla CSS** - Inline styles (no CSS framework)
- **Vanilla JavaScript** - No frameworks (React, Vue, etc.)

## External Libraries
- **Tabler Icons** - Icon library via CDN
  - URL: https://cdn.jsdelivr.net/npm/@tabler/icons-webfont@latest/tabler-icons.min.css
- **Google Fonts - Inter** - Font family via Google Fonts
  - URL: https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap

## Backend/API
- **Anthropic Claude API** - AI chatbot backend
  - Model: claude-sonnet-4-6
  - Endpoint: https://api.anthropic.com/v1/messages

## Architecture
- Single-file HTML application
- No build tools or bundlers
- No backend server required
- Direct API calls to Anthropic Claude
